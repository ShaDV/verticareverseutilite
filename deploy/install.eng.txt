SQL Reversing for Vertica

# Required
OS: Windows 7 or above, Linux with X-server, MacOS
RAM: minimum 1 gb
HDD: minimum 35 mb
JRE: version 1.8 or above

# Install program for Linux or MacOS
1. Unpack zip file to directory and go to this dir
2. Set execution right for file reversing.sh
3. Run reversing.sh

# Install program for Windows
1. Unpack zip file to directory and go to this dir
3. Run reversing.bat
