SQL Reversing for Vertica

# Требования
OS: Windows 7 или выше, Linux с X-сервером, MacOS
RAM: минимум 1 гигабайт
HDD: минимум 35 мегабайт
JRE: версия 1.8 или выше

# Установка программы для Linux или MacOS
1. Разверните архив в директорию и перейдите в эту директорию
2. Установите права на выполнение для файла reversing.sh
3. Запустите reversing.sh

# Установка программы для Windows
1. Разверните архив в директорию и перейдите в эту директорию
2. Запустите reversing.bat
